# DART Transit Token Page

모바일 전용 DART 교통 티켓 페이지

## 설치 방법

```bash
npm install
```

## 실행 방법

```bash
npm start
```

브라우저에서 http://localhost:3000 으로 접속하세요.

## DART 로고 추가하기

`src/assets/` 폴더에 실제 DART 로고 이미지를 추가해주세요.
- PNG 파일을 사용하려면: `dart-logo.png`로 저장하고 `App.js`의 import를 수정하세요
- SVG 파일을 사용하려면: 현재 placeholder `dart-logo.svg`를 교체하세요

## 빌드

```bash
npm run build
```
